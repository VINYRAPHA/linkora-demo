# Maquette Linkora

Première maquette éditoriale sans vente ni stockage distant. Elle permet de valider la direction visuelle, le vocabulaire, la collection initiale et l’expérience de démonstration.

Ouvrir `index.html` dans un navigateur pour la consulter.

La page `create.html` permet de composer gratuitement une capsule. Le texte est conservé dans le stockage local du navigateur. Les médias restent en mémoire pendant la session et ne quittent jamais l’appareil. Cette version ne permet donc pas encore de consulter la capsule depuis un autre téléphone.

Le visuel principal est conceptuel et doit être remplacé par une photographie produit avant commercialisation.
